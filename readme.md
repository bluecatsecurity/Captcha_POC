# Captcha POC

Simple POC of google reCaptcha using html/js frontend and nodeJS backend


## Installation

Just git clone 

```bash
Install Nodejs
Install express
Install axios
Install nodemon
```

## Usage

```python
node server.js
nodemon server.js
```
